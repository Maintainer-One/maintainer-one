# Welcome to Maintainer One

Maintainer One is a sports simulation powered by open-source logic. This manual covers the core concepts of the game, the simulation engine, and how to write your first team script.

## Core Concepts

- **Ticks**: The game runs in discrete time steps called ticks.
- **Positions**: The field is a grid where players and point zones are positioned.
- **Captures**: Players score points by occupying point zones.
- **Stuns**: Players can stun opponents to prevent them from moving or scoring.
